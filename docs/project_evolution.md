# Project Evolution

This project is the result of a progression from manual data handling to automated analytics.

## Early Work
- manually compiled IPL ball-by-ball data using Cricsheet
- updated datasets match-by-match
- built Tableau dashboards for analysis

## Limitations
- manual process was time-consuming
- difficult to scale across seasons
- no automation

## Current Project
- automated data collection using Python
- scraped match data from ESPN Cricinfo
- built a scalable pipeline for data transformation and analysis

This evolution reflects growth from manual workflows to building structured data systems.
