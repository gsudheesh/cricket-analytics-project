## Current Analysis (2025)

These visuals are generated from the automated data pipeline built in this project.

They reflect:
- ball-by-ball level analysis
- scalable data processing using Python
- deeper contextual insights compared to earlier dashboards

### Key views

- **Run Rate Trends**  
  Evolution of scoring patterns across matches and phases

- **Batting Patterns**  
  Player-level performance and scoring behavior

- **Bowling Patterns**  
  Wicket-taking trends, economy and control metrics

- **Situational Analysis**  
  Performance across match contexts (phases, pressure situations)

These outputs demonstrate the transition from manual dashboards to automated, data-driven analysis workflows.
